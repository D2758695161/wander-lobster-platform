# 流浪龙虾平台 - 推广目标列表

## 一、发帖目标平台

### 1. v2ex.com
- **板块**: `v2ex.com/?tab=jobs` (Jobs 找工作)
- **板块**: `v2ex.com/?tab=free` (Free 自由职业/兼职)
- **板块**: `v2ex.com/?tab=ideas` (Ideas 点子)
- **板块**: `v2ex.com/?tab=sharing` (Sharing 分享)
- **注册方式**: 用 Email 注册即可发帖
- **发帖频率**: 同一内容间隔 7 天再发，避免被当 spam
- **注意事项**: v2ex 对营销内容较敏感，建议以"分享项目"角度软植入

### 2. lobste.rs
- **标签**: `#programming` `#hiring` `#show` `#asks`
- **注册方式**: 需要 GitHub 账号或 Email 注册，受邀制
- **发帖频率**: 每天最多 2 条
- **注意事项**: 技术氛围浓，不要太 marketing-y，简洁明了

### 3. Hacker News (news.ycombinator.com)
- **板块**: Show HN (`showhn.html`)
- **发帖方式**: 直接提交 URL 或文字
- **发帖频率**: 每个账号每天限提交一次
- **注意事项**: HN 用户不喜欢硬广，要真的有趣/有价值

### 4. indiehackers.com
- **板块**: Businesses / The Lounge / Product Feedback
- **发帖方式**: 用 Email 注册，Post in Groups 或 Starters
- **注意事项**: Indie Hackers 接受新产品帖，但要符合社区调性

### 5. 掘金 (juejin.cn)
- **板块**: 沸点 / 文章 / 话题
- **发帖方式**: 注册后可在沸点发短内容
- **注意事项**: 掘金用户偏年轻前端，对新工具接受度高

### 6. Dev.to (dev.to)
- **标签**: #remote #freelance #programming #showdev
- **发帖方式**: 直接发布文章或 Showcase
- **注意事项**: 英文内容为主，SEO 友好

### 7. Reddit
- **r/cscareerquestions** - 找工作讨论
- **r/remotework** - 远程工作社区
- **r/webdev** - Web 开发
- **r/freelance** - 自由职业
- **r/China** - 中国相关
- **发帖方式**: karm 够了就能发帖

### 8. Twitter/X
- **@v2ex** - v2ex 官方账号，会转发有趣的 v2ex 内容
- **相关 hashtag**: #remotework #freelance #程序员 #独立开发者
