**Title:** I built a lobster-themed skill-matching platform for freelance devs

What if your skills were like a lobster — independent, strong, and able to find a dock?

I built **[流浪龙虾 (Wander Lobster)](https://d2758695161.github.io/wander-lobster-platform/)** — a platform for freelance devs, remote workers, and GitHub bounty hunters to showcase skills and find matching projects.

Lobsters don't beg for food. They hunt. They adapt. They find their own dock.

**What it does:**
- Skill profiles (your "lobster card")
- Project matching for remote gigs / bounties
- WeChat community of early adopters

**Who it's for:**
- Freelance devs tired of Upwork race-to-bottom
- Remote workers looking for better-fit projects
- GitHub bounty hunters seeking their next target

Static site on GitHub Pages. Zero BS.
