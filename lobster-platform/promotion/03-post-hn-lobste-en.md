# Show HN: Wander Lobster — A Skill-Matching Platform for Freelance Developers

**What if your skills were like a lobster — independent, strong, and able to find a dock?**

I built [流浪龙虾 (Wander Lobster)](https://d2758695161.github.io/wander-lobster-platform/) — a platform for freelance devs, remote workers, and GitHub bounty hunters to showcase skills and find matching projects.

**The hook:** Lobsters don't beg for food. They hunt. They adapt. They find their own dock. That's the vibe we're going for.

**What it does:**
- Skill profiles for devs (think: your lobster card)
- Project matching for remote gigs / bounties
- WeChat community of early adopters (QR code on the site)

**Who it's for:**
- Freelance developers tired of Upwork race-to-bottom
- Remote workers looking for better-fit projects  
- GitHub bounty hunters seeking their next target

**Tech stack:** It's a static site (GitHub Pages) — lightweight, fast, no BS.

Link: https://d2758695161.github.io/wander-lobster-platform/

Happy to hear feedback. Looking for fellow lobsters 🦞
