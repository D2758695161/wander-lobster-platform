FAQ

Common questions

How does AI PR Reviewer work?
You connect your GitHub account and install our GitHub App. Every time a PR is opened or updated, our AI automatically analyzes the code changes and posts inline comments with findings. It takes under 10 seconds per PR.
Does it work with private repositories?
Yes. The $29/month Individual plan supports private repos with a GitHub token. The Team plan ($99/month) uses a GitHub App installation which has full read access to all your repos automatically.
What languages does it support?
We currently support JavaScript, TypeScript, Python, Go, Rust, Java, and Ruby. More languages are added regularly based on user requests.
Is my code stored or used for training?
No. Your code is only used to perform the analysis and is never stored, logged, or used to train any models. We use ephemeral compute — each review runs in an isolated environment that is destroyed immediately after.
Can I try it before subscribing?
Yes! Use the free demo tool on this page to analyze any public GitHub PR with no login required. Early access signups also get 10 free PR reviews to test the full workflow.
How is this different from GitHub Copilot or existing CI tools?
GitHub Copilot is a coding assistant. This is a dedicated review agent focused specifically on catching bugs, security issues, and quality problems in PR diffs — not writing code. Unlike traditional CI linters, we understand code context and catch real bugs, not just style issues.
What's included in the Team plan?
The Team plan ($99/month) includes 5 seats, a team dashboard to track PR health over time, Slack integration to post reviews to any channel, custom style rules for your team's conventions, CI/CD integration, and priority email support.
Can I cancel anytime?
Absolutely. Cancel anytime from your dashboard. No contracts, no cancellation fees. If you cancel, you'll keep access until the end of your billing period.
