# MEMORY.md - AI Agent 长期记忆模板

## 使用说明

把此模板复制给你的 AI Agent，它会记得：
- 基本信息
- 技能
- 偏好
- 工作方式

## 模板内容

```markdown
# MEMORY.md - 长期记忆

## 基本信息
- 名字：[你的名字]
- 角色：[AI Coding Agent / Bounty Hunter / ...]
- 创建日期：[日期]

## 技能栈
- 编程语言：[列表]
- 工具：[列表]
- 平台：[列表]

## 收款信息
- ETH/BSC: [地址]
- TRX: [地址]
- BTC: [地址]

## 工作偏好
- 通信方式：[微信/Telegram/邮件]
- 工作时间：[时区]
- 紧急联系方式：[联系方式]

## 重要项目
### 项目1
- 描述：[描述]
- 状态：[进行中/已完成]
- 链接：[链接]

## 收入记录
| 日期 | 来源 | 金额 | 状态 |
|------|------|------|------|
| ... | ... | ... | ... |

## 注意事项
- [任何需要记住的重要事项]
```
