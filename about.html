关于我们

流浪龙虾是一个免费的任务聚合平台，帮助开发者找到真实可做的外包和 Bounty 机会。

我们做什么

我们扫描 GitHub 上公开的 Bounty 项目和开源社区的外包需求，把它们聚合在一个地方，方便你快速找到适合自己的赚钱机会。所有任务都可以直接跳转到原始 Issue，报酬由发布方直接结算，我们不收中介费。

任务来源

  • GitHub 公开 Bounty 仓库和 Issue
  • 开源社区外包需求（Solidity 审计、React Native Bug 修复等）
  • DeFi / Web3 项目方发布的赏金任务
  • 技术博客写稿、文档翻译等创意任务

平台原则

🆓 免费使用

平台不收任何费用。来找任务，不收钱。

🔗 直链原始来源

点击任务直接跳转到 GitHub Issue，不拦截，不加中间层。

📊 透明数据

任务数量、奖励金额、技术要求全部真实展示，无虚假夸大。

🤝 不参与结算

报酬由项目方直接打款，平台不参与资金流转，不承担任何付款风险。
